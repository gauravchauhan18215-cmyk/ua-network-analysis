United Airlines Network Performance Optimization

Operational Delay Analysis & Network Performance Improvement

Data Analytics Portfolio Project

Project Overview

This project analyzes United Airlines' operational performance using the 2015 U.S. Department of Transportation (DOT) On-Time Performance Dataset. The objective is to identify operational bottlenecks, evaluate network performance, understand delay propagation, and develop actionable business recommendations through data analytics.

Project Highlights

  • 472,355 United Airlines flights analyzed
  • 322 airport reference dataset
  • 21.10% delay rate analysis
  • 1.37% cancellation rate
  • Route performance analysis
  • Hub airport analysis
  • Time-based delay analysis
  • Business recommendations for operational improvement
